## Kelly Windows Data
https://github.com/jhkweb/Kelly-Windows-Data/pull/7
### PR Classification
Serialization consistency update.

### PR Summary
This PR replaces the custom `FlexibleBoolConverter` with a new `BoolJsonConverter` using Newtonsoft.Json, standardizing boolean JSON serialization across model records and updating repository logic.

- Adds `BoolJsonConverter` and replaces all `[JsonConverter(typeof(FlexibleBoolConverter))]` attributes with `[JsonConverter(typeof(BoolJsonConverter))]` in model records.
- Updates `JsonFileRepository<T>` to derive file names from type and always use Newtonsoft.Json settings.
- Updates global usings to include `Newtonsoft.Json` and remove `System.Text.Json.Serialization`.
- Removes redundant `using` statements throughout the codebase.
- Adds new `ComponentProductCheck` record using the new converter.
