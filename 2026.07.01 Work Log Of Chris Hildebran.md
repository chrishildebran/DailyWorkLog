## Kelly Windows Logic
https://github.com/jhkweb/Kelly-Windows-Logic/pull/4
### PR Classification

New feature and utility extension methods for numeric, image, and object handling.

### PR Summary

This pull request introduces new extension methods for doubles, images, and objects, and adds a utility class for fraction calculations.

- DoubleExtensions.cs: Added `AsFraction`, `IsOutOfRange`, and multiple `ToPercentage` methods for enhanced numeric formatting and validation.
